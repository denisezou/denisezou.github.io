What the project does: This project generates a letter for the user to guess in 7 tries. 
Why the project is useful: This project is useful because it demonstrates basic skills in Javascript for the evening coding course for adults.
How users can get started with the project: Users may get started with the project by pressing letter keys on the keyboard to guess which letter the computer chose. 
Where users can get help with your project: You may help with my project on Github by following "denisezou".
Who maintains and contributes to the project: I maintain the project with the help of my tutor, Leah Daniels, and the teaching assistants.
